# Khasm

a concern library that does what Mixin can't

## Using

```groovy
repositories {
    maven {
        url 'https://maven.concern.i.ng/'
    }
}

dependencies {
    modImplementation include('khasm:khasm:1.1.0')
}
```

Get the latest version (instead of `1.1.0`) from the [properties file](gradle.properties)
